Password To Passkey

Login/Password + Save Passkey On This Device

This dedicated client demo shows the transition flow: the user enters username/password, completes password login, and then saves a passkey on a real device using Windows Hello (Face/PIN).

The password is not stored in the passkey. A passkey contains a cryptographic key stored in the device platform authenticator.
